# Автоматизированная информационная система семантического поиска

Данный репозиторий содержит исходный код курсовой работы, посвященной разработке системы семантического поиска в документации, связанной с организацией учебного процесса.

## Технологический стек
* **Платформа:** .NET 8 / C#
* **Архитектура:** Многослойная клиент-серверная архитектура
* **Веб-фреймворк:** ASP.NET Core MVC
* **База данных:** Microsoft SQL Server LocalDB
* **ORM:** Entity Framework Core
* **Тестирование:** xUnit
* **Фронтенд:** HTML5, CSS3, JavaScript (Fetch API, асинхронный Progress Bar)

## Структура репозитория
* `SemanticSearch/` — исходный код веб-приложения (Контроллеры, Сервисы, Репозитории, Модели).
* `SemanticTest/` — пакет автоматизированных тестов.
* `Docs/` — папка с файлом пояснительной записки.
